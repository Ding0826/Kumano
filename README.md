# 深度導覽（GUIDE_DATA）

請加入歷史沿革補充

## 熊野本宮大社
介紹：
熊野三山之主祭神社，古代熊野信仰核心。平安時代起，上皇與貴族頻繁參拜形成「熊野詣」潮流。  
歷史沿革：原址位於大齋原，明治22年水患後遷至現址。  

冷知識：
1. 主祭神為家都美御子大神。  
2. 古時稱為「日本第一大靈驗所」。  
3. 黑色八咫烏為熊野象徵。

---

## 大齋原
介紹：
熊野本宮舊社地，位於熊野川沙洲。  
歷史沿革：1889年洪水沖毀社殿後成為聖域遺址。

冷知識：
1. 擁有日本最大級鳥居之一。  
2. 曾為多王子社集中地。  
3. 被列入2004年世界文化遺產。

---

## 那智瀑布
介紹：
日本最高單一落差瀑布（133m），為飛瀧神社神體。  
歷史沿革：自古即為修驗道修行場。

冷知識：
1. 每年7月舉行那智火祭。  
2. 與青岸渡寺形成神佛習合景觀。  
3. 被列入世界遺產。
